# FuelUp

## Design

Shared design reference for the FuelUp mobile web app.

### How to view

Open **[`fuelup-design-reference.html`](fuelup-design-reference.html)** in any browser
(double-click the file, or drag it into a browser tab). No build step, no dependencies.

The **Landing screen is interactive** — hover a half to preview-expand, click to enter,
click the center FuelUp pill to reset.


## Tech Stack (per PRD)

React + Vite · Tailwind CSS + shadcn/ui · Space Grotesk + Inter · Supabase · Vercel
