04 · Landing High fidelity

Landing — “Split Duo”

A 50/50 split entry that visually states FuelUp's two equal sides — donor (cyan) and creator (violet) — echoing the "AI Equality" theme. Interactive: hover a half to preview-expand, click to enter (Expand & Enter transition), click the center FuelUp pill to reset. Try it below.

Notes

  • Inline SVG line icons (give-heart / idea-bulb), not emoji.
  • Glassmorphism logo pill floating over a glowing divider.
  • Soft radial glows — cyan top, violet bottom — for the "two equal sides" metaphor.
  • Transition: Expand & Enter (selected half fills, loading state).
  • Built with pure CSS (hover + hidden-radio state machine) — no JS, smooth on screen recording.
  • Real 393pt width with Dynamic Island & home-indicator safe areas.

07 · Platform

Mobile & safe-area spec

Designed for modern phones (iPhone 17 Pro class). Build flexible, not pixel-locked.

  • Base logical width 390–402pt (Pro class); Pro Max ≈ 430pt. Layouts flex across this range.
  • Reference height ≈ 852–874pt.
  • Respect safe areas with env(safe-area-inset-*) + viewport-fit=cover: top Dynamic Island ≈ 59pt, bottom home indicator ≈ 34pt.
  • Keep primary actions out of the safe-area zones; use relative units, flex, and min/max sizing.
